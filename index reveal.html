Цель проекта:

анализ популярных веб-фреимворков

для создания презентаций

    Задачи проекта:

  • Поиск популярных веб-фреимворков


  • Изучение возможностей фреимворков impress.js,reveal.js


  • Создание презентации на одном из фреимворков


  • Сравнительный анализ фреиворков, в том числе сравнение с популярными решениями для создания презентаций (Prezi,Power Point,Google Slides)

Почему веб-фреимворки?

Создание презентации с помощью кода (для любителей писать код)

Более легкая работа в текстовых редакторах, совместная работа и отслеживание изменений в тексте


Удобнее размещать и показывать в онлайн (для html необходим только браузер)

Устройство reveal.js и impress.js


  • html файл (презентация)


  • css файл (заранее настроенные стили)


  • js файлы (код на языке java script,обработчики событий и анимации)

  • gif
  • вставка формул

  • смена стиля презентации
  • древовидная структура презентации
